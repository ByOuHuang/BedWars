
## 介绍
###### 这是一个高仿Hypixel起床战争小游戏的插件，由一个16岁的宅男制作，
###### 制作他的原因仅仅是因为网上的BedWars插件都已经跟不上时代了...233

#

##### Java版本:
###### | Java 8 (推荐Java 11)

##### Minecraft版本:
###### | Spigot-1.8.9


##### 数据储存方式:
###### | MySQL


##### 插件依赖：
###### | ProtocolLib, Citizens
